Since vertical strips are parallel to the \(y\) axis we can find \(I_y\) by evaluating this integral with \(dA = y\ dx\text{,}\) and substituting \(\frac{h}{b} x\) for \(y\)

\begin{align*} I_y \amp = \int_A x^2\ dA\\ \amp = \int_0^b x^2\ y\ dx\\ \amp = \int_0^b x^2 \left (\frac{h}{b} x \right ) dx\\ \amp = \frac{h}{b} \int_0^b x^3 dx\\ \amp = \frac{h}{b} \left . \frac{x^4}{4} \right\vert_0^b\\ I_y \amp = \frac{hb^3}{4}\text{.} \end{align*}

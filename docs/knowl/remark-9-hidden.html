Recall that the first moment of area \(Q_x = \int \bar{x}_{el}\ dA\) is the distance weighted area as measured from a desired axis. The distance term \(\bar{x}_{el}\) is the the distance from the desired axis to the centroid of each differential element of area, \(dA\text{.}\)

If you're using a single integral with a vertical element \(dA\text{,}\) as shown in the left image below,

\begin{equation*} dA = \underset{height}{y(x)}\ \underset{base}{dx} \end{equation*}

and the horizontal distance from the \(y\) axis to the centroid of \(dA\) would simply be

\begin{equation*} \bar{x}_{el}=x \end{equation*}

It is also possible to find \(\bar{x}\) using a horizontal element, as shown in the right image below, but the computations are a bit more challenging. First the equation for \(dA\) changes to

\begin{equation*} dA = \underset{base}{x(y)}\ \underset{ height}{(dy)} \end{equation*}

Additionally, the distance to the centroid of each element, \(\bar{x}_{el}\) , must measure to the middle of the horizontal element. For this triangle,

\begin{equation*} \bar{x}_{el}=\frac{x(y)}{2} \end{equation*}

We find a similar contrast to finding the vertical centroidal distance \(\bar{y}\) where it is easier to use a \(dy\) element to find \(\bar{y}\) than it is to use a \(dx\) element.

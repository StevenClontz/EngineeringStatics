Item 3. Combine the pieces to find the overall centroid.

After you have filled in the whole table, you can find the coordinates of the centroid by applying (7.5.1) with the summary values from the last row.

\begin{align*} \bar{x} \amp = \frac{Q_y}{A} \amp = \frac{112.8}{24.93} \amp = \cm{4.52}\\ \bar{y} \amp = \frac{Q_x}{A} \amp = \frac{41.86}{24.93} \amp = \cm{1.692} \end{align*}

Finally, plot the centroid \((\bar{x}, \bar{y})\) on the diagram. If you have made a calculation error it will usually be obvious, because the centroid location won't “feel right.”

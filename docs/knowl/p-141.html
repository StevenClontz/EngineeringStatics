Steps to create a unit vector of vector \(\vec{F}\text{.}\)

  1. Define the vector in component form

    \begin{equation*} \vec{F} = \left \langle F_x, F_y \right \rangle \end{equation*}

  2. Compute the magnitude of the vector using the Pythagorean Theorem

    \begin{equation*} F = |\vec{F}| = \sqrt {{F_x}^2 + {F_y}^2} \end{equation*}

  3. Divide the components by the overall length

    \begin{equation*} \widehat{\vec{F}}=\frac{\vec{F}}{|\vec{F}|} = \frac{\left \langle F_x, F_y \right \rangle}{F} \end{equation*}

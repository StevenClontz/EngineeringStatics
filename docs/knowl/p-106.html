The easiest way to handle vector subtraction is to add the negative of the vector you are subtracting to the other vector. In this way, you can still use the tip-to-tail technique after flipping the vector you are subtracting. You can also simulate this in FigureÂ 2.1.1 by following the steps below.

  1. Find \(\vec{A}-\vec{B}\) where \(\vec{A}=\cm{2}\) and \(\vec{B}=\cm{3}\text{.}\)
  2. Set \(\vec{A}\) to a value of \(\cm{2}\) and \(\vec{B}\) to a value of \(\cm{-3}\) (the negative of its actual value).
  3. Slide the tail of either vector to find the resultant (or sum) of these two vectors. The order does not matter because vector addition is commutative.
  4. The result should be \(\cm{-1}\text{.}\)

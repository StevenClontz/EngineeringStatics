Item 4. Apply the equations of equilibrium.
\begin{align*} \Sigma \vec{F} \amp = 0 \qquad \begin{cases} \Sigma F_x = 0 : \amp C_x + F_x = 0 \\ \amp C_x = - \lb{218} \\ \Sigma T_y = 0 :\amp C_y - F_y = 0 \\ \amp C_y = + \lb{436} \\ \Sigma T_z = 0 : \amp C_z - F_z = 0 \\ \amp C_z = + \lb{109} \\ \end{cases}\\ \\ \Sigma \vec{M} \amp = 0 \qquad \begin{cases} \Sigma M_x = 0 :\amp M_x + {M_C}_x = 0 \\ \amp M_x = \ftlb{-2180} \\ \Sigma M_y = 0 : \amp M_y + {M_C}_y = 0 \\ \amp M_y = \ftlb{-1530} \\ \Sigma M_z = 0 : \amp M_z +{M_C}_z = 0 \\ \amp M_z = \ftlb{+1750} \\ \end{cases} \end{align*}

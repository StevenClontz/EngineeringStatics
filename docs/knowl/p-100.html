The standard notation for a vector uses either an arrow above the vector name or the vector name in bold font. All three of these notations mean the same thing.
\begin{equation*} \vecarrow{F} = \bar{F} = \vec{F} \end{equation*}
Most printed works including this book will use a bold font for vectors, but for handwritten work you and your instructor will use the bar or arrow notation.

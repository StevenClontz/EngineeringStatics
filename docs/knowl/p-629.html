Particle equilibrium requires that \(\sum \vec{F} = 0\text{,}\) so,

\begin{equation*} A\ \hat{\vec{A}} + B\ \hat{\vec{B}} + C\ \hat{\vec{C}} = - W\ \hat{\vec{W}}\text{.} \end{equation*}

This is a 3 \(\times\) 3 system of three simultaneous equation, one for each coordinate direction, which needs to be solved for \(A\text{,}\) \(B\text{,}\) and \(C\text{.}\)

\begin{align*} A \cos \ang{35} - B \cos \ang{15} \cos \ang{30} + 0 \amp = 0 \amp \amp \amp (\Sigma F_x \amp= 0)\\ A \cos \ang{55} + B \cos \ang{30} + C \cos \ang{70} \amp = \N{196.2} \amp \amp \amp (\Sigma F_y \amp= 0)\\ 0 - \cos \ang{15} \cos \ang{60} + C \cos \ang{20} \amp = 0 \amp \amp \amp (\Sigma F_x \amp= 0) \end{align*}

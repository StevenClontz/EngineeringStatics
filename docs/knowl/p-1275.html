Begin by identifying the bounding functions. From the diagram, the boundaries are 1) the spandrel, 2) the \(x\) axis and, 3) the vertical line \(x = b\text{.}\) In the function \(y=kx^n\text{,}\) constant \(k\) has not been specified, but it is not arbitrary. The diagram indicates that the function passes through point \((a,b)\text{.}\) There is only one value of \(k\) which will cause this, and we can find it by substituting \(a\) and \(b\) into the function for \(x\) and \(y\) then solving for \(k\text{.}\)

\begin{align*} y \amp = k x^n\\ b \amp = k a^n\\ k \amp = \frac{b}{a^n} \end{align*}

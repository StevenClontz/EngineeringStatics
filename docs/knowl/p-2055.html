Finding \(I_x\) using horizontal strips is anything but easy. In fact, the integral that needs to be solved is this monstrosity

\begin{align*} I_x \amp = \int_A y^2\ (1-x)\ dy\\ \amp = \int_0^2 y^2 \left (1- \frac{\sqrt[3]{2} \left ( \sqrt{81 y^2 + 12} + 9y \right )^{2/3} - 2 \sqrt[3]{3}}{6^{2/3} \sqrt[3]{\sqrt{81 y^2 + 12} + 9y}} \right )\ dy\\ \amp \dots \text{ and then a miracle occurs}\\ I_x \amp = \frac{49}{120}\text{.} \end{align*}

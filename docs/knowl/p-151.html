For nearly all three-dimensional computations, you need the \(x\text{,}\) \(y\text{,}\) and \(z\) components of each vector before proceeding with the computations. If you are given the components upfront, then you are set to move forward, but sometimes vectors are defined by either direction cosine angles or spherical angles. The next sections detail how you can move back and forth between these angular systems and \(x\text{,}\) \(y\text{,}\) and \(z\) components.

Extending two-dimensional Cartesian coordinates, you can express any point or vector in three-dimensional space with components in the \(x\text{,}\) \(y\text{,}\) and \(z\) orthogonal directions. The notation remains the same as for two-dimensional vectors, with the addition of a \(z\)-component vector and the corresponding unit vector \(\khat\text{.}\)

More formally, if \(a\) is a set with \(n\) elements, then the average, or mean value is

\begin{equation} \bar{a} = \frac{1}{n} \sum_{i=1}^n a_i = \frac{a_1 + a_2 + \dots + a_n}{n} \text{.}\tag{7.1.1} \end{equation}

The bar over the \(a\) indicates that this is a mean value. This average is also called the arithmetic mean. When calculating an arithmetic mean, each number is equally important when evaluating the average.

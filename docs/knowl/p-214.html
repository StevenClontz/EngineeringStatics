Finally, the last component, the angle \(\phi\text{,}\) is measured down from the \(+z\) axis to the vector \(\vec{F}\text{.}\) Using a right triangle between \(\vec{F}\) and the \(z\) axis, we find \(\phi\) using the inverse cosine \(\cos^{-1}\) of the adjacent side, \(F_z\text{,}\) divided by the hypotenuse \(|\vec{F}|\text{.}\) Recognize that \(\phi\) is the same angle as the direction cosine angle \(\theta_z\text{.}\)

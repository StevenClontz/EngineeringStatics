If a unit vector is multiplied by a scalar value, the vector is 'scaled' by that amount, so for instance if \(\ihat\) is multiplied by \(100~\text{ N} \) the result is a \(100~\text{ N} \) vector pointing to the right, if \(\jhat\) is multiplied by \(-50~\text{ N} \) the result is a \(50~\text{ N} \) vector pointing down.

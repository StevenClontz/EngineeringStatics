Find the neutral axis.

For one L\(8\times 4\times 1/2\) angle, from SectionÂ C.1

\begin{align*} A_\text{L} \amp = \inch{4.75}^2\\ \bar{y}_\text{L} \amp = \inch{1.98}\\ \bar{I}_\text{L} \amp = \inch{17.3}^4\text{.} \end{align*}

For the \(b = \inch{8},\) \(h = \inch{1}\) rectangle

\begin{align*} A_\text{R} \amp = bh = \inch{8}^2\\ \bar{y}_\text{R} \amp =-h/2 = \inch{-0.5}\\ \bar{I}_\text{R} \amp = \frac{bh^3}{12} = \frac{8}{12} = \inch{6.67}^4\text{.} \end{align*}

Find the distance to the neutral axis

\begin{align*} \bar{y} \amp = \frac{\sum A_i \bar{y_i}}{\sum A_i} = \frac{ 2 A_\text{L} \bar{y}_\text{L} + A_\text{R} \bar{y}_\text{R} } {2 A_\text{L} + A_\text{R}}\\ \amp = \frac{ 2 (4.75)(1.98) + (4)(-0.5)} { 2(4.75) + 4}\\ \bar{y} \amp = \inch{1.245}\text{.} \end{align*}
